\begin{SCIONglossary}

  \gitem{Path Server}{Facility within an isolation domain where end
  hosts register their up- and down-paths. End hosts query a path
  server in order to obtain valid paths do a destination.}

  \gitem{Isolation Domain (ISD)}{Hierarchical and self-contained unit with
  own jurisdiction in which a number of ASes are located.}

\end{SCIONglossary}
